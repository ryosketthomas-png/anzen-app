# 安全書類管理システム

大和ライフネクスト専門工事向け 安全書類デジタル管理アプリ

## 機能一覧

- 新規入場者登録（新規入場教育資料閲覧 → 6ステップ入力 → 手書きサイン → QR発行）
- 入退場記録（検索式作業員選択・工種別集計）
- 安全日誌・KY指示書（AI安全対策自動生成・チェックリスト・承認フロー）
- 承認フロー（担当者チェック → 主任・上司が後追い承認）
- 案件登録（元請管理者 / 協力会社入力を分離）

## セットアップ

1. このリポジトリをGitHubにpush
2. Settings → Pages → Source: main branch / root
3. 発行されたURLをスマホで開く
